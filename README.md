# resize-instagram
resize for instagram photo online without cropping

[![resize instagram](https://github.com/xncbf/resize-instagram/blob/master/app/static/resize-instagram.png)](https://resiz.io/)

# Usage:sunglasses:

 1. Upload images
 2. Select option
 3. Click download
 4. Enjoy!:clap::clap::clap:
